環境設定

General 説明
背景 画像の後ろに表示する色や模様を設定できます。
フルスクリーン フルスクリーンモードで使う画面を指定します。
ほかの画面を暗くする フルスクリーンモード中にほかのディスプレーを暗くします。
画像を大きくしたらアンチエイリアスする 元の大きさより画像を大きくしたらアンチエイリアスします。小さくした場合はいつもアンチエイリアスされます。
Escape キー Escape キーの動きを設定できます。
進み方 説明
マウスボタンのクリック マウスボタンをクリックしたときの動きを指定できます。“次のページに行く” は設定された読み方の方向によって右や左になります。それに対して、“左のページに行く” と “右のページに行く” は読み方の方向によらずそれぞれの方向へ移動します。副クリックは主クリックと反対の方向へ移動します。
逆に進んだ時の位置 前の画像に進んだ場合に画像のどこを表示するかを指定できます。
フォルダに入る程度 何回フォルダの中のフォルダに入るか設定する。
ソフトウェア更新: 説明
自動で確認する 週に一回自動で更新がないか確認します。更新がある場合、Sequential メニュー > 更新をインストールする... を選択してインストールすることができます。これにはインターネット接続が必要です。
