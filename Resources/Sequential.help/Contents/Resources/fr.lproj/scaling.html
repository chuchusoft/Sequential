Scaling modes

Mode Description
Taille actuelle Displays images normally with no scaling.
Ajustement automatique Redimensionne les images pour qu'elles soient aussi larges que possible tout en minimisant les défilements inutiles.
Ajuter à la fenêtre/ l'écran Scales images so that no scrolling is required.

For the Automatic Fit and Fit to Window/Screen modes, you can choose whether to allow images to be upscaled, downscaled, or scaled freely (both). Resolution independent images (such as PDFs) as always scaled freely.

You can also choose to apply a constant scaling factor.
